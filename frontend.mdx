Software development essentially entails the side of web development that entails user interactivity. It basically offers users an interface where they can communicate with the website or mobile app.

Frontend languages are:

- HTML
- CSS
- JavaScript
- TypeScript

Frontend frameworks include
- Tailwind CSS(A CSS library)
- ReactJs (A JavaScript library)
- Next Js (A web framework based on React ReactJs)

Here is a code snipet in TypeScript
```ts
 const sayHello = (name: string): string => {
   console.log("Hello ", name)
 }
 sayHello("Tony") // Hello Tony
```
